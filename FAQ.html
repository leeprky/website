Frequently Asked Questions

How Do I Get Updates?

POWER CORD

Powercord will automatically keep your files up to date if they havent been modified, if so they will try to update but you will need to press force update to overide any changes you have made.

BETTER DISCORD

On BetterDiscord my themes have a automatic banner telling you if your theme is outdated and you can reinstall it from the link provided or on my website as they will always be the latest version on my website. If not download it from the raw link as shown how in Themes.

How Do I Install A Theme?
BETTERDISCORD

Download The File Either From The Website, The BetterDiscord Site OR The Raw File On The Github
Then Place It In The Dirrectory Below For Windows

POWERCORD

For Powercord You Can Use Terminal/CMD

cd powercord/src/Powercord/themes && git clone (LINK)

Replacing (LINK) With The URL Like 
https://github.com/leeprky/Bloop

Downloading The Version As A Zip Means You Wont Get Auto Updates!

How Do I Edit A Theme?

POWER CORD

On Powercord Locate The Theme File:

C:\Users\name\powercord\src\Powercord\themes\

And open the config file usually named and inside it read what you have to as each theme is different and it usually just involves changing the numbers or removing /* only from the beginning of @import making it simple to change the theme.

BETTER DISCORD

On Betterdiscord go to the themes area in settings and over the theme you would like to edit and click the little edit button and it will open the theme for you to make changes.

How Do I Install Glasscord?

POWER CORD

Not OPTIONAL FOR BETTERDISCORD ( USE SETTING CALLED TRANSPARENCY IN BD )

Step 1:

Glasscord.asar within the themes glasscord-files folder. (Given in my themes main folder)

Move glasscord.asar to C:\Users[PCNAME]\AppData\Local\DiscordCanary\app-1.0.36\resources\app.

Step 2:

Open The index.js in C:\Users[PCNAME]\AppData\Local\DiscordCanary\app-1.0.36\resources\app. "require('C:/Users/[PCNAME]/AppData/Local/DiscordCanary/app-1.0.42/resources/app/glasscord.asar')" - Change [PCNAME] To your name in this line. CTRL+S Or Save The File

Step 3

Insiide the theme of your choice that supports it remove the import for the glasscord import or for some themes remove the /* of the glasscord section and then refresh discord via task manager and your theme should be transparent

Optionally: https://github.com/AryToNeX/Glasscord/wiki/Installation
