SO.. YOUR INTERESTED IN CREATING YOUR OWN THEME(S)?

Below I'll teach you in a few simple tutorials making use of my tools and a few handle CSS tips and tricks which is what you use to create themes within discord and websites like this one!

learn more

STEP ONE

First Signup To Github if you have not already. Then on the the homepage click create new and you should be greated with the create a new repository screen. Create a name and a description if you choose public.
Add a readme file | Add a licience and select which one you like (Recommded "MIT") and then click create.
You can also use a template, I have Made One! (Link Here)

STEP TWO

Now We Need To create a few new files.

We Name The first file " [THEME-NAME].css " [THEME-NAME] being whatever you want to call your theme. And then just commit new file at the bottom.

Secondly we create a file called " powercord_manifest.json " exactly how it is

STEP THREE

Now We Need To Click On " powercord_manifest.json " in any text editor of choice.
I Highly recommend Visual Studio Code as its one of the best editors.

Once opened this file we need to input the following:

{ "name": "[THEMENAME]", 
description": "DESCRIPTION TEST", 
"version": "0.0.1", 
"author": "Discord#1234", 
"license": "MIT", 
"theme": "theme-name.css", }
