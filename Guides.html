SO.. YOUR INTERESTED IN CREATING YOUR OWN THEME(S)?

Below I'll teach you in a few simple tutorials making use of my tools and a few handle CSS tips and tricks which is what you use to create themes within discord and websites like this one!

STEP ONE

First Signup To Github if you have not already. Then on the the homepage click create new and you should be greated with the create a new repository screen. Create a name and a description if you choose public.
Add a readme file | Add a licience and select which one you like (Recommded "MIT") and then click create.
You can also use a template, I have Made One! (Link Here)

Need Help

STEP TWO

Now We Need To create a few new files.

We Name The first file " [THEME-NAME].css " [THEME-NAME] being whatever you want to call your theme. And then just commit new file at the bottom.

Secondly we create a file called " powercord_manifest.json " exactly how it is

Need Help

STEP THREE

Now We Need To Click On " powercord_manifest.json " in any text editor of choice.
I Highly recommend Visual Studio Code as its one of the best editors.

Once opened this file we need to input the following:

{ "name": "[THEMENAME]", 
description": "DESCRIPTION TEST", 
"version": "0.0.1", 
"author": "Discord#1234", 
"license": "MIT", 
"theme": "theme-name.css", }

Need Help

Basicially You Are Done...​ To Install it you would go to the link in github and type into CMD/Terminal
cd powercord/src/Powercord/themes && git clone https://github.com/name/theme-name

Im Going To Now Explain what i would do now if I was creating a new theme and teaching you advanced features.

Learn CTRL+SHIFT+I Which is your best friend now

STEP FOUR

Now at this step i would usually add some extra files to support betterdiscord and glasscord.
So we will create a folder called betterdiscord and within that folder we name it " [THEME-NAME].theme.css " making sure we include .theme.css as better discord requires it

Need Help

STEP FIVE

Now Inside the betterdiscord theme file the one we just created we will put the following

/** 
* @name test-theme 
* @author name#1234 
* @version 0.0.1 
* @description Desc 
* @source https://github.com/name/theme-name 
* @updateUrl [THE RAW URL (LEFT CLICK THIS FILE AND CLICK VIEW RAW)] 
* @authorId discord user id 
* @invite discord invite
* @donate donation links 
* @website a website 
*/

STEP SIX

We are now ready to begin editing our main css file.

So everything i have made following the same formatting and i use my website to import these files which is simple to setup and provide near instant updates to your themes without the need for reinstalling the theme...

On The First Or Second Line I Would Include A @import which is importing another CSS file from the internet.

@import "https://leeprky.github.io/themes/theme-name/imports.css";

^^ This will just import that user and inside that URL is all my theme tools like alt colour, fully themed discord and greenple.
